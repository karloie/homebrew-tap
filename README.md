# 🍺 homebrew-tap

Custom Homebrew tap for karloie projects.

## Installation

Tap this repository:

```bash
brew tap karloie/tap
```

## Available Casks

### [kompass](https://github.com/karloie/kompass)

[![GitHub release](https://img.shields.io/github/v/release/karloie/kompass)](https://github.com/karloie/kompass/releases)
[![License](https://img.shields.io/github/license/karloie/kompass)](https://github.com/karloie/kompass/blob/main/LICENSE)

Kubernetes cluster visualization and exploration tool.

```bash
brew install karloie/tap/kompass
```

### [bastille](https://github.com/karloie/bastille)

[![GitHub release](https://img.shields.io/github/v/release/karloie/bastille)](https://github.com/karloie/bastille/releases)
[![License](https://img.shields.io/github/license/karloie/bastille)](https://github.com/karloie/bastille/blob/main/LICENSE)

```bash
brew install karloie/tap/bastille
```

## Usage

After installation, the binaries will be available in your PATH:

```bash
kompass --help
bastille --help
```

## Updating

To update to the latest versions:

```bash
brew update
brew upgrade kompass
brew upgrade bastille
```
